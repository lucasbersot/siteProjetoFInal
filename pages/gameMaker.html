QUANDO E ONDE FOI CRIADO?

O GameMaker Studio 2 foi lançado oficialmente no dia 8 de março de 2017, pela YoYo Games, como uma atualização e evolução do GameMaker: Studio (GMS1).

TRAJETÓRIA

O desenvolvimento da ferramenta, inicialmente chamado de Animo, começou em 1999 com Mark Overmars, mas foi a partir de 2007, com a incorporação na YoYo Games, que o projeto ganhou mais força e se tornou o GameMaker: Studio, culminando no lançamento do GameMaker Studio 2.

GAMEMAKER STUDIO 2: FOCO EM 2D E FACILIDADE

é um motor de jogo popular, especialmente conhecido por sua abordagem amigável para iniciantes e excelência no desenvolvimento de jogos 2D.

MELHOR PARA QUAIS JOGOS?

Jogos 2D: Sua principal força. Ideal para pixel art, plataforma, RPGs 2D, top-down e uma vasta gama de jogos bidimensionais (ex: Undertale, Hotline Miami, Shovel Knight). Prototipagem Rápida: Permite criar e testar ideias de jogos de forma muito ágil. Projetos Indie: É uma ferramenta favorita entre desenvolvedores independentes e pequenos estúdios. Jogos Multiplataforma: Suporta exportação para PC, mobile e consoles.

LINGUAGEM DE PROGRAMAÇÃO

Predominantemente GameMaker Language (GML): Uma linguagem de script própria, otimizada para o motor, com sintaxe similar a C ou JavaScript, projetada para ser fácil de aprender. Oferece GML Visual: Um sistema de arrastar e soltar (Drag and Drop - DnD) para criar lógica de jogo sem escrever código, ideal para iniciantes.

MERCADO DE TRABALHO

Nicho Específico: Embora não tenha o mesmo volume de vagas que Unity ou Unreal, há um mercado para desenvolvedores GMS2, especialmente em estúdios menores e para projetos 2D. Foco em Portfólio: É uma ótima ferramenta para iniciantes construírem um portfólio de jogos finalizados, o que é crucial para entrar na indústria. Habilidades Transferíveis: O aprendizado de GML e os conceitos de desenvolvimento de jogos em GMS2 são transferíveis para outras engines e linguagens. Trabalho Freelancer: Existem oportunidades para trabalho freelancer e projetos por contrato utilizando GMS2.
